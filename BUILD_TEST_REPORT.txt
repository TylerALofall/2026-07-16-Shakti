## Fixed loader defect

The first runnable demo originally stopped because the strict tablet loader required spoken sound files for Foundation punctuation even though Level 0 defines those sounds as optional.

The loader now:

- requires text, written text, and visual art for Foundation 0,
- does not block Foundation loading on deferred punctuation audio,
- continues to require validated sound for counting and alphabet tablets.

The ledger and runtime now apply the same rule.
