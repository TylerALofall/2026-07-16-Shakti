## THREE FILES EVERY MODEL MUST READ
1. `Genesis/Eden/README.md`
2. `Genesis/Eden/LAWS.md`
3. `Genesis/Eden/MODEL_INSTRUCTIONS.md`

Read all three before changing anything.

1. **NO PYTHON.** Not for generation, editing, testing, moving, or renaming.
2. **NO VECTORS, EMBEDDINGS, OR PROBABILISTIC SELECTION.** Absolute mathematics stays absolute.
3. **NO SELECTIVE RUNTIME LOADING.** Approved sorted tables become resident at build time. A run manifest selects resident rows.
4. **NO MANUAL LINK GRAPH.** Compute relationships through classification joins and conversion tables.
5. **CLEAN HOUSE.** Do not bulk-import the old forge. Only correct, complete files enter.
6. **STABLE DATA; EDITABLE TABLETS.** Signed data paths never move and carry no dates. A tablet XML may be corrected in place only when `last_modified`, its index entry, and the ledger are updated.
7. **ONE TABLET, ONE OBJECT.** Each stone contains exactly `visual_text`, `visual_art`, `audio_art`, and `text`, in that order.
8. **CALLER OWNS TRAINING ORDER.** XML declares direct relevant prerequisites only. It never schedules repetition or practice. The calling script follows `INDEX_MASTER.xml` and the prerequisite graph.
9. **ONE-LINE ORDER.** Curriculum order exists only in `INDEX_MASTER.xml`.
10. **SIGNED INDEX.** Every entry names the file, submitter, `date_added`, and `last_modified`. Pending tablets do not enter the curriculum.
11. **ONE PATH FACT.** `file_path` is one complete repository-relative filename and location. Do not split or duplicate it.
12. **FIXED LOAD SLOTS.** The load enum is defined once in C. Empty optional elements preserve their slot; tablets never carry their own enum.
13. **VISUAL TEXT IS DERIVED.** The frozen renderer makes a sequence of `uint64_t` 8x8 glyphs. `visual_art` is only an optional real picture.
14. **COUNTING PICTURE.** Show numeral, word, quoted word, and exact marks. Fill rows of ten; ten rows make a hundred square. Compute it from the value.
15. **CONVERGENCE IS NOT FORCED AGREEMENT.** Present the channels as one moment. Silence or background content is not a contradiction unless it directly conflicts with the focus.
16. **TIMING IS CALLER-OWNED.** Tablet XML has no timing fields. Initial values 1–10 receive spans of 1–10 seconds. Rules for 11–1000 must be signed before use.
17. **LEVELS.** Level 0 is ASCII 32–127 pre-exposure; Level 1 is counting 1–1000; Level 2 is ABCs. Counting and ABCs may have no prerequisite.
18. **TRUTH VALUES.** Answers are TRUE, FALSE, or UNKNOWN. UNKNOWN names what is missing.
19. **AUDIO MUST CARRY SIGNAL.** Every referenced WAV passes the approved signal check.
20. **RECEIPTS.** Claims require inspectable output. A successful compile alone is not proof.
21. **FIRST AND LAST.** Read the three mandatory files first. Sign every exact changed path last; wildcards are refused.
